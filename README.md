#Arrayception

Given an array of arbitrarily nested arrays, return n, where n is the deepest level that contains a non-array value.

Examples:

Input ----> Output

array:

[ [ 5 ], [ [ ] ] ] ----> Result 2

array:

[ 10, 20, 30, 40 ] ----> Result 1

array:

[ [ 10, 20 ], [ [ 30, [ 40 ] ] ] ] ----> Result 4

array:

[ ] ----> Result 0

array:

[ [ [ ] ] ] ----> Result 0
